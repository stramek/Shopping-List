# Shopping List

## Introduction
Simple application wrote fully in Kotlin that allows to create and manage shopping lists.

## Used popular libraries and technologies
- Kotlin language
- MVP architecture
- Dagger2
- RxJava2
- Room Persistence
- JUnit4
